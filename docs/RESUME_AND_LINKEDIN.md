## LinkedIn project description

CryoBedNet is a computational sustainability project focused on Antarctic bed-topography super-resolution. It reimplements the DeepBedMap idea in a modern PyTorch workflow and adds region holdout evaluation, HPC training scripts, baseline comparisons, terrain-aware losses, and a dashboard for exploring predictions and residuals.

The project connects machine learning, geospatial data processing, cryosphere science, and climate-risk visualization.
